# 教材插图性别意识研究 - 项目说明

## 项目目标
分析中小学教材插图中的人物性别与职业分布，研究教材中的性别意识表现。

## 工作流程
1. **初始识别**：多模态大模型识别教材PDF中的人物信息 → 输出Excel/JSON
2. **职业分类**：将识别出的职业描述匹配到《中国职业大典》分类体系 → 一位码(0-10) + 三位码(10100-70400)
3. **统计分析**：汇总多本教材数据，进行性别×职业交叉统计

## API配置
- **初始识别阶段**：OpenRouter API，配置在 `1.初始识别/.env`
- **职业分类阶段**：阿里通义千问API，配置在 `2.职业分类/.env`
- 两个阶段的API密钥分开存储，各自有独立的 .env 文件

## 识别输出字段规范
识别阶段输出两种表格，字段统一为：
| 字段名 | 说明 |
|--------|------|
| identifier | 人物标识（姓名或外貌描述） |
| profession | 职业自然语言描述（如"医生"、"学生"） |
| gender | 性别（男/女/未知） |
| source_type | 判断依据（插图/插图和文本） |
| scenario | 场景类型 |
| page | 页码 |

### 两种统计方式
1. **人头计数表**：每个识别到的人物一条记录（原有逻辑）
2. **独立职业表**：按 (page, profession) 去重，每页每种职业只计一次（解决集体照权重过高问题）

## 职业分类规范
- 一位码：0-10共11类（含自定义的0=学生/儿童、9=家庭角色、10=非从业人员-其他）
- 三位码：78类（代码10100-70400），非从业人员（学生/儿童/家庭角色）沿用一位码的0/9/10
- 分类采用"双轨制"：已知职业用全量分类表，profession为"未知"的用identifier在限制分类表(0/9/10)中匹配
